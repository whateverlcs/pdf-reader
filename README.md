Uma simples ferramenta que permite os usuários a lerem PDFs com mais facilidade, ao selecionar uma pasta, será identificado todos os PDFs identificados nela, possibilitando a leitura entre eles.

## Como usar

1. Clone o repositório: `git clone https://github.com/whateverlcs/pdf-reader`.
2. Navegue até o diretório da solução: `cd PDFReader`.
3. Realize o Build e rode a solução.

## Features

- Leitura de PDFs identificados em uma pasta usando o leitor de pdf chromium
- Interface User-Friendly
